配置OpenGL環境

安装GLM库

從這裡下載GLEW，放入OpenGL工程所在目錄，並include之。

安装GLEW库

從這裡下載GLEW，放入OpenGL工程所在目錄，並include之。

移动\bin\Release\Win32\glew32.dll到C:\Windows\System32和C:\Windows\SysWOW64中。

移動\include\GL\glew.h和\include\GL\glxew.h以及\include\GL\wglew.h到C:\Program Files (x86)\Windows Kits\8.0\Include\um\gl中。

移動\lib\Release\Win32\glew32.lib和\lib\Release\Win32\glew32s.lib到C:\Program Files (x86)\Microsoft Visual Studio 11.0\VC\lib中。

Link Error及解決方案

打開项目菜单 - 属性 - 配置属性 - 链接器 - 输入 - 附加依赖项，在其中增加glew32.lib。

程序前面加入：

#pragma comment(lib,"glew32.lib")

采用release方式编译

将项目 - 配置属性 - C/C++ - 代码生成 - 运行时库改为：单线程调式（/MLd)
