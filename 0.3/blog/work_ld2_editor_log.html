製作Lingoes辭典

任務名稱 Lingoes辭典ld2格式破解
任務比重 輕
創建日期 2017.05.02
最後更新 2017.05.03

Lingoes的辭典真的很好用，我都把它設置成開機自動啟動了，可見他對我這種英文渣渣的使用頻率有多高。但是，自己寫一個屬於自己的辭典就不容易了。Lingoes似乎並不想對外公開自己的接口，所以，我很無奈的開始了我漫長的破解工作。

需求開始

因為我想自己創建一個只屬於我自己的辭典，但是又不想用 Source File Format of Lingoes Dictionary 中，Please send your finished dictionary source file to kevin-yau@msn.com, Lingoes will compile it into .ld2 for you.這樣子的方法，所以我決定自己寫一個ld2格式編輯器。

話說你們的編輯器要研發到什麼時候啊？所以，讓我來幫你們做吧。

給kevin-yau@msn.com的郵件

雖然說我很不喜歡讓別人幫我做事情，但是，我還是給lingoes發了一封郵件，裡面有我自己創建的一份辭典。

不知到能不能收到他們給我編譯好的ld2文件。也不知道我有沒有能力破解他們的編譯方式。

目前，已經過去近一個星期，仍然沒有收到回覆。

版權問題

一个有关“Lingoes灵格斯电子词典LD2(LDF)文件解析(附java词典导出程序)”的想法中ld2 binary format is copyright protected and is not open for free projects.這句話真讓我很尷尬。
