About Windows OS

我覺得不需要太多的介紹了吧。這年頭，已經很少有人知道不是windows操作系統的電腦了。而作為主流操作系統，其問題卻可以說是，如滔滔江水。我對windows系統的研究，也有幾年了。現在我使用的是windows 8.1 x64。

Windows常用快捷鍵

前段時間，我在同學電腦上表演了一波windows的純鍵盤操作，讓那同學看得眼花撩亂。那同學叫我整理一下自己所懂的快捷鍵，好讓他也能夠如此的牛逼。
其實，這些快捷鍵是我打開電腦後經常使用的，應該來說是使用給自己看，節省自己使用滑鼠的頻率，從而節省自己的生命。說真的，那天如此牛逼的表演，也讓我自己有點吃驚。

以下，以#表示Win，^表示ctrl，+表示Shift，!表示Alt

windows系統指令

key describe
#L 鎖屏
#E 打開資源管理器
#R 運行
#C 查看時鐘，打開右邊欄 ｢windows8/8.1 專享｣
#left 左移當前窗口
#right 右移窗口
#up 最大化當前窗口
!Tab 切換當前運行程式
!+Tab 逆向切換當前運行程式

windows程式通用指令

key describe
^Tab 切換標籤
^+Tab 逆向切換標籤
^W 關閉當前標籤
^+T 重新打開最近關閉的標籤
^O 打開
^S 保存
^+S 另保存為 ｢有些軟體不支援｣
^N 創建
^A 全部選擇
^X 剪下
^V 貼上
^Z 撤銷
^F 查找
^H 替換
^G 定位
^P 打印
del 刪除
insert 插入/改寫模式切換
F1 幫助
F2 重新命名
F5 刷新
F11 全屏
!F4 關閉窗口

DOS程式指令

key describe
!X 退出進程
break 中斷進程
+del cut
^insert copy
+insert paste

Run中常用指令

key describe
services 服務
regedit 註冊表
cmd 命令提示字元
notepad 記事本
dxdiag DirectX診斷
calc 計算器

Run中不常用的指令

key describe
gpedit 策略組
dcomcnfg 系統服務
winver 系統版本信息
osk 屏幕鍵盤 ｢系統自帶的，非常垃圾，｣
psr 步驟紀錄器 ｢非常牛逼的軟體，但是我沒怎麼用過｣
msconfig 系統設置
control 控制面板
powershell 類似命令提示字元的工具
secpol 本地安全政策 ｢基本沒用過｣
mstsc 遠程連結

上面的工具，是我曾經使用過，保留在我Run中的指令。
